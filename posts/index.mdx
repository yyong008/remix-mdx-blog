# Welcome to Remix Mdx Blog!

> A light blog usage remix with mdx file.

## how to write ?

cd `/posts` dir, and create a `xxx.mdx` fileï¼Œnow you can write you mdx blog with markdown.

## usage

```sh
# clone
git clone https://github.com/yyong008/remix-mdx-blog.git

# install
pnpm i

# run
pnpm run dev

# open
# open your browser `http://localhost:3000/` and enjoy it!

# build
pnpm run build

# start
pnpm run start
```

## stack

- Remix V2 run JavaScript/TypeScript
- MDX buil your markdown and aritcle
- Tailwindcss handle you css
